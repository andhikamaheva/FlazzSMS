# FlazzSMS

**FlazzSMS** merupakan aplikasi OpenSource yang dibangun menggunakan Laravel 5.2. **FlazzSMS** memungkinkan pengguna untuk mengirimkan sms ke satu tujuan atau banyak tujuan dalam satu waktu.
